![Banner](Docs/banner.png)

Un Escape Game immersif en réalité virtuelle dans lequel vous explorez les mystères d'une grotte.

https://github.com/user-attachments/assets/71431232-b454-44c8-bbc7-db53a229df34 


## Installer le build Android
1. Télécharger le build depuis [la dernière release](https://github.com/mataquet/VR_EscapeGame/releases/latest)
2. Copier le `.apk` sur votre appareil Android
3. Installer et lancer le jeu en VR

## Tester dans Unity (éditeur)
1. Cloner le repo :  
```bash
git clone https://github.com/mataquet/VR_EscapeGame.git
cd VR_EscapeGame
```

2. Ouvrir le projet avec Unity Hub
3. Ouvrir la scène VR_EscapeGame/Assets/Branches/dev/dev.unity
4. Activer le XR Simulator
5. Lancer le jeu (Play) et se déplacer en maintenant Tab
